## Aim:

  To study the Pandas library and understand its basic functionalities such as creating Series and DataFrames, data manipulation, 

  and performing basic statistical analysis.

## Theory

  Pandas is an open-source Python library used for data manipulation and analysis. It is especially useful for handling structured data.

## Key Concepts:

### Series:

  A one-dimensional labeled array capable of holding data of any type (integers, strings, etc.).
  
### DataFrame:

  A two-dimensional data structure (like a table) consisting of rows and columns.
  
## Operations Performed:

  ### 1.Creation of Series and DataFrame
  
  -Series is created using a list.
  
  -DataFrame is created using a dictionary.
  
  ### 2.Understanding Structure
  
  -shape gives dimensions (rows, columns)
  
  -size gives total number of elements
  
  -ndim gives number of dimensions
  
  -columns shows column names
  
  ### 3.Data Access
  
  -Column access using column name
  
  -Row access using loc and iloc
  
  -Viewing data using head() and tail()
  
  ### 4.Data Modification
  
  -Adding new columns
  
  -Updating values using indexing
  
  -Deleting columns using drop()
  
  ### 5.Statistical Analysis
  
  -Mean, minimum, and maximum values of a column
  
  ### 6.Filtering
  
  -Applying conditions to extract specific rows (e.g., marks > 80)
